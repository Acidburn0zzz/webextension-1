Ace Script
=============

Userscript manager based on [Violentmonkey](https://github.com/violentmonkey/violentmonkey)

Build
---
``` sh
# Install dependencies
$ yarn

# Build for Firefox
$ ./build.sh firefox

# Build for Chrome
$ ./build.sh chrome
```

Build was tested with:
* Ubuntu Ubuntu 14.04.4 LTS
* yarn 1.12.3
* node 8.12.0

The result will be placed to 'xpi/' folder.
